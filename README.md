# 🤖 Python Command-Line Assistant

A simple command-line assistant built with Python that helps perform common daily tasks directly from the terminal. This project was created to practice Python fundamentals, including functions, file handling, loops, conditional statements, modules, and user interaction.

---

## 🚀 Features

* 👋 Greeting command
* 🧮 Built-in calculator
* 📝 Save notes to a text file
* 📖 Read saved notes
* 🔤 String utilities

  * Length
  * Uppercase
  * Lowercase
  * Reverse string
  * Find a word
* 📂 Open folders from the terminal
* 📁 Display files in the current directory
* 📜 Command history
* 🎮 Number guessing game
* ❓ Built-in help menu

---

## 🛠️ Technologies Used

* Python 3
* os module

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/Python-Command-Line-Assistant.git
```

2. Open the project folder:

```bash
cd Python-Command-Line-Assistant
```

3. Run the program:

```bash
python assistant.py
```

## 📋 Available Commands

| Command      | Description                         |
| ------------ | ----------------------------------- |
| greet        | Display a greeting                  |
| calculate    | Perform basic arithmetic            |
| save note    | Save a note to a file               |
| read note    | Read saved notes                    |
| string tools | Perform string operations           |
| open folder  | Open a folder using its path        |
| show files   | List files in the current directory |
| history      | Display command history             |
| game         | Play a number guessing game         |
| command help | Show all available commands         |
| quit         | Exit the assistant                  |
